# FocusLedger

FocusLedger is a Pomodoro-based study and focus management application designed to help users structure study sessions, manage schedules, and track productivity.

## Features

- Pomodoro session timer
- Adaptive schedule management
- IST clock integration
- Session tracking & analytics
- Daily dashboard & progress monitoring
- Custom audio alerts
- Notes & activity tracking

## Tech Stack

- React.js
- LocalStorage (Persistence Layer)
- Modern CSS Design System

## Getting Started

npm install  
npm start
